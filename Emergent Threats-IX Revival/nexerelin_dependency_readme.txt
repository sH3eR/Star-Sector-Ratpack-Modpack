The "undo nexerelin dependency" folder let you switch between making this mod playable without Nexerelin, or making Nexerelin a hard dependency while enabling alliance features for the IX Battlegroup and Trinity Worlds. To use:

1). Go into the preferred sub-folder and copy/paste both "mod_info.txt" and the "data" folder to the IX Revival main mod directory, answer yes when asked to overwrite files
2). Delete "mod_info.json" in the main directory
3). Rename "mod_info.txt" to "mod_info.json"
